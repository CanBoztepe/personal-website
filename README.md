## Project Setup

Make sure you have [Node.js] and [npm] installed.

[Node.js]: https://nodejs.org/
[npm]: https://www.npmjs.com/

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
